# dpc-words

DPC (診断群分類) 関連のデータをまとめたリポジトリです。ICD コード (国際疾病分類) や日付付きの DPC リストを提供しています。

## データ

- [icd.csv](icd.csv) ICD (国際疾病分類) のデータ
- [icd_list.csv](icd_list.csv) 日付付きの ICD リスト

データは毎日自動で更新されます。
