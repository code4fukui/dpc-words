# dpc-words

DPC(診断群分類)関連のデータをまとめたリポジトリです。ICDコード(国際疾病分類)や日付付きのDPCリストを提供しています。

## サンプルアプリ

- [DPCリスト](https://code4fukui.github.io/dpc-words/)

## データ

- [icd.csv](icd.csv) ICD(国際疾病分類)のデータ
- [icd_list.csv](icd_list.csv) 日付付きのICDリスト

データは毎日自動で更新されます。
